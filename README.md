# AI Daily Digest 🤖

> 每日自动收集 AI 领域优质创作者的推文，生成结构化报告

[![Auto Update](https://img.shields.io/badge/auto-update-brightgreen)](https://github.com/SeanLiangYoung/ai-daily-digest)
[![Daily](https://img.shields.io/badge/frequency-daily-blue)](https://github.com/SeanLiangYoung/ai-daily-digest)

## 📚 关于

这个项目自动追踪 AI 领域的优质创作者在 X (Twitter) 上的动态，每日生成精选内容摘要。

**目标：**
- 📊 追踪 AI 行业最新动态
- 🔥 发现优质内容和讨论
- 📝 整理成结构化、易读的报告
- 🤖 完全自动化，由 AI 助手 Kai 维护

## 🎯 追踪的创作者

本项目追踪来自以下领域的优质创作者：

完整列表见：[creators.json](creators.json)

## 🛠️ 技术栈

- **数据收集：** X API / Web 抓取
- **报告生成：** TypeScript + Bun
- **自动化：** OpenClaw Cron
- **存储：** GitHub Repository
- **通知：** Telegram

## 📊 报告格式

每日报告包含：
- 📈 今日概览（推文数、主要话题）
- 🔥 热门内容精选
- 📝 分类整理（技术突破、产品发布、行业观点等）
- 🔗 完整创作者列表
